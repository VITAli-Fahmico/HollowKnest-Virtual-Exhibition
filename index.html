About Hollow Knight

Hollow Knight is a 2D epic adventure game made by studio Team Cherry. Set in the mysterious, underground world of Hallownest. The main character is a small but brave bug - going by the same name as the game itself - exploring ancient ruins, battling strange creatures and uncovering hidden secrets.
The game offers a rich and unforgettable journey through its beautiful hand-drawn art and atmospheric storytelling.
